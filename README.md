# LMS

## Requirements

- Install MongoDB
- Install Node.js
- Install Yarn Package

## Quick start

- Install dependencies: `npm install` or `yarn install` (Yarn Recommended)
- Start the server: `npm run start:dev` or `yarn start:dev`
- DEV URL: `localhost:3001`
